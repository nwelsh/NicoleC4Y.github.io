Lessons 1 and 2

In lesson 1, we begin by learning intro to the web and basic HTML, including learning about HTML tags, learning the sections, and creating the first HTML page.
At the end of lesson one, the student will make a homework assinment with the following requirements:

  1. 3 headings - any size
  2. 1 image
  3. 2 images that do not work
  4. 4 paragraphs
