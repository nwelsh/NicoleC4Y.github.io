Lessons 5 and 6

In lesson 1, we begin by learning intro to the web and basic HTML, including learning about HTML tags, learning the sections, and creating the first HTML page.
At the end of lesson one, the student will make a homework assinment with the following requirements:

1. 3 headings - any size
2. 1 image
3. 2 images that do not work
4. 4 paragraphs

Link to Chapter 1

In lesson 2, we learn attributes fr links and images, text formatting, special characters and comments.
At the end of lesson two, the student will make a homework assignment with the following requirements:

1. Heading as the title/bolded
2. Write a paragraph
3. Use Hr tag to create a line
4. Special symbols paragraph

Link to Chapter 2
After lessons 1 and 2, the students get work time on the homework assignments and do the quiz.
