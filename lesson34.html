Lessons 3 and 4

In lesson 3, we learn about block and inline elements (span and div) and advanced hyperlinks.
At the end of lesson three, the student will make a homework assignment with the following requirements using this google doc :

1. Go to link
2. Copy the about and home page into 2 HTML docs
3. Add HTML code

Link to Chapter 3

In lesson 4, we learn to present data with lists and tables.
At the end of lesson four, the student will make a homework assignment with the following requirements:

1. Ranked foods with bolded title, ordered list
2. random foods, unordered lsit
3. definition lists
4. table with anything, images

After lessons 3 and 4, the students get work time on the homework assignments and do the quiz. The midterm is then introduced.
